# 数据模型与存储

## 1. 数据库概览

- **数据库**：MySQL (`xiaozhi_esp32_server`)
- **连接池**：Druid
- **ORM**：MyBatis-Plus
- **迁移**：Liquibase (`db/changelog/db.changelog-master.yaml`)
- **缓存**：Redis

## 2. 核心数据表

## 4. Redis 缓存策略

### 4.1 会话/临时数据

| Key模式 | 用途 | TTL |
|---------|------|-----|
| `captcha:{uuid}` | 图形验证码 | 5分钟 |
| `sms:code:{phone}` | 短信验证码 | 5分钟 |
| `sms:interval:{phone}` | 发送间隔 | 60秒 |
| `sms:count:{phone}:{date}` | 每日发送次数 | 当天 |
| `ota:download:{uuid}` | OTA下载映射 | 限次 |
| `audio:play:{uuid}` | 音频播放映射 | 短期 |
| `tmp:register:{mac}` | 临时注册 | 5分钟 |
| `device:activation:{code}` | 激活码映射 | 限期 |

### 4.2 缓存数据

| Key模式 | 用途 | 更新时机 |
|---------|------|---------|
| `server:config` | 全站配置树 | 配置变更时 |
| `model:name:{id}` | 模型名称 | 模型更新时 |
| `model:config:{id}` | 模型配置 | 模型更新时 |
| `timbre:name:{id}` | 音色名称 | 音色更新时 |
| `timbre:details:{id}` | 音色详情 | 音色更新时 |
| `agent:devices:{agentId}` | 智能体设备数 | 设备变更时 |
| `dict:{type}` | 字典数据 | 字典更新时 |

### 4.3 Python 侧缓存

| 缓存类型 | 用途 | 机制 |
|---------|------|------|
| `CacheType.CONFIG` | 主配置 | LRU + TTL |
| OTA固件扫描 | 固件文件列表 | TTL |
| Provider实例 | 各AI模型实例 | 全局单例 |
