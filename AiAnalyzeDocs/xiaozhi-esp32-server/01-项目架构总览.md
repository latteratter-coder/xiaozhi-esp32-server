# 小智 ESP32 服务端 - 项目架构总览

## 1. 项目简介

小智 ESP32 服务端（xiaozhi-esp32-server）是开源硬件 [xiaozhi-esp32](https://github.com/78/xiaozhi-esp32) 的后端服务，按照小智通信协议，使用 **Python、Java、Vue** 三种主要技术实现。支持 MQTT+UDP、WebSocket、MCP、声纹识别、知识库等能力。

## 3. 四大核心模块

| 模块 | 路径 | 技术栈 | 端口 | 职责 |
|------|------|--------|------|------|
| **xiaozhi-server** | `main/xiaozhi-server/` | Python 3.10, asyncio, websockets, aiohttp | 8000 (WS) / 8003 (HTTP) | 设备对话核心服务：WebSocket通信、AI管线（ASR/LLM/TTS/VAD）、插件系统、MCP |
| **manager-api** | `main/manager-api/` | Java 21, Spring Boot 3.4.3, MyBatis-Plus, Shiro | 8002 | 智控台REST API：用户/设备/智能体/模型/知识库/OTA管理 |
| **manager-web** | `main/manager-web/` | Vue 2.6, Element UI, Vuex, vue-i18n | 开发端口自定 | PC端智控台Web界面 |
| **manager-mobile** | `main/manager-mobile/` | uni-app, Vue 3, Vite, TypeScript, Pinia | 移动端 | 移动端智控台应用 |

## 5. 部署架构

### 5.1 最简部署

仅部署 `xiaozhi-server`（Python），本地配置 `config.yaml`，不依赖 manager-api。

### 5.2 全模块部署

通过 `docker-compose_all.yml` 部署完整栈：

```
xiaozhi-esp32-server (Python 8000/8003)
  ├── manager-web + manager-api (Nginx + JRE, 8002)
  ├── MySQL
  └── Redis
```

Docker镜像构建：
- `Dockerfile-server-base`：Python基础镜像（ffmpeg、libopus、pip依赖）
- `Dockerfile-server`：Python服务镜像
- `Dockerfile-web`：多阶段构建（Node构建Vue + Maven打包Java + Nginx+JRE运行）

## 7. 配置体系

| 配置文件                   | 位置                       | 作用                                  |
| ---------------------- | ------------------------ | ----------------------------------- |
| `config.yaml`          | `main/xiaozhi-server/`   | Python服务主配置（端口、日志、AI模型等）            |
| `data/.config.yaml`    | 用户本地创建                   | 覆盖默认配置（不入版本库）                       |
| `config_from_api.yaml` | `main/xiaozhi-server/`   | 从智控台拉配置模板                           |
| `application.yml`      | `manager-api/resources/` | Spring Boot主配置（端口8002、context-path） |
| `application-dev.yml`  | `manager-api/resources/` | 开发环境（MySQL、Redis连接）                 |
| `.env.*`               | `manager-web/`           | 前端环境变量                              |
| `env/.env.*`           | `manager-mobile/`        | 移动端环境变量                             |

**配置优先级**：`data/.config.yaml` > `config.yaml`；智控台模式下由 manager-api 管理配置，本地多数项失效。

## 8. 配置项详细说明

> 以下配置仅在非智控台模式下生效。使用智控台时，请在智控台界面修改。

### 8.1 服务器基本配置 (server)

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `server.ip` | string | `0.0.0.0` | 监听地址，`0.0.0.0` 监听所有网卡 |
| `server.port` | int | `8000` | WebSocket 服务端口 |
| `server.http_port` | int | `8003` | HTTP 服务端口（OTA、视觉分析） |
| `server.websocket` | string | 自动生成 | OTA 返回给设备的 WebSocket 地址，Docker/公网部署需手动配置 |
| `server.vision_explain` | string | 自动生成 | 视觉分析接口地址 |
| `server.timezone_offset` | string | `+8` | OTA 返回信息的时区偏移量 |
| `server.mqtt_gateway` | string | null | MQTT 网关地址 |
| `server.mqtt_signature_key` | string | null | MQTT 签名密钥 |

**认证配置 (server.auth)**

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `server.auth.enabled` | bool | `false` | 是否启用认证 |
| `server.auth.allowed_devices` | list | `[]` | 设备白名单 MAC 地址列表 |

### 8.2 运行时配置

| 配置项 | 类型 | 默认值 | 说明 |
|--------|------|--------|------|
| `delete_audio` | bool | `true` | 使用完音频文件后是否删除 |
| `close_connection_no_voice_time` | int | `120` | 无语音输入多久后断开连接（秒） |
| `tts_timeout` | int | `10` | TTS 请求超时时间（秒） |
| `tool_call_timeout` | int | `30` | 工具调用超时时间（秒） |
| `enable_wakeup_words_response_cache` | bool | `true` | 唤醒词加速缓存 |
| `enable_greeting` | bool | `true` | 开场是否回复唤醒词 |
| `enable_stop_tts_notify` | bool | `false` | 说完话是否开启提示音 |
| `enable_websocket_ping` | bool | `false` | WebSocket 心跳保活 |
| `tts_audio_send_delay` | int | `0` | TTS 音频发送延迟 |

### 8.3 音频参数 (xiaozhi.audio_params)

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| `format` | `opus` | 音频格式 |
| `sample_rate` | `24000` | 采样率，支持 8000/12000/16000/24000/48000 |
| `channels` | `1` | 声道数 |
| `frame_duration` | `60` | 帧时长（毫秒） |

### 8.4 模块选择 (selected_module)

| 配置项 | 可选值 |
|--------|--------|
| `VAD` | `SileroVAD` |
| `ASR` | `FunASR`, `DoubaoASR`, `AliyunASR` 等 |
| `LLM` | `ChatGLMLLM`, `DoubaoLLM`, `DeepSeekLLM` 等 |
| `VLLM` | `ChatGLMVLLM`, `QwenVLVLLM` |
| `TTS` | `EdgeTTS`, `DoubaoTTS`, `AliyunTTS` 等 |
| `Memory` | `nomem`, `mem0ai`, `powermem`, `mem_local_short` |
| `Intent` | `nointent`, `intent_llm`, `function_call` |

### 8.5 LLM 配置

```yaml
LLM:
  ChatGLMLLM:
    type: openai
    base_url: https://open.bigmodel.cn/api/paas/v4/
    model_name: glm-4-flash
    api_key: your_api_key
    temperature: 0.7
    max_tokens: 500
```

通用参数：`type`、`base_url`/`url`、`model_name`、`api_key`、`temperature`、`max_tokens`、`top_p`、`frequency_penalty`

支持的 LLM 提供商：阿里云百炼(AliLLM)、火山引擎(DoubaoLLM)、DeepSeek、智谱AI(ChatGLMLLM)、Ollama、Dify、Gemini、Coze、百度千帆(QianfanCodeLLM)

### 8.6 TTS 配置

```yaml
TTS:
  EdgeTTS:
    type: edge
    voice: zh-CN-XiaoxiaoNeural
    output_dir: tmp/
```

支持的 TTS 提供商：EdgeTTS(免费)、DoubaoTTS、HuoshanDoubleStreamTTS、AliyunTTS、TencentTTS、OpenAITTS、CosyVoiceSiliconflow、FishSpeech、GPT_SOVITS_V2/V3

### 8.7 ASR 配置

FunASR本地：`type: fun_local`, `model_dir`, `output_dir`
FunASR服务：`type: fun_server`, `host`, `port`
DoubaoASR：`type: doubao`, `appid`, `access_token`, `cluster`

其他：DoubaoStreamASR、TencentASR、AliyunASR、BaiduASR、OpenaiASR、GroqASR、VoskASR、XunfeiStreamASR、Qwen3ASRFlash

### 8.8 VAD 配置

```yaml
VAD:
  SileroVAD:
    type: silero
    threshold: 0.5
    model_dir: models/snakers4_silero-vad
```

### 8.9 Intent 配置

- `nointent`：不使用意图识别
- `intent_llm`：使用独立LLM提前分析意图，`type: intent_llm`, `llm`, `functions`
- `function_call`：使用主LLM的function calling，`type: function_call`, `functions`

### 8.10 Memory 配置

- `nomem`：不使用记忆
- `mem_local_short`：本地短期记忆，`type: mem_local_short`, `llm`（用于总结的LLM）
- `mem0ai`：云端记忆，`type: mem0ai`, `api_key`
- `powermem`：企业级记忆，`type: powermem`, `enable_user_profile`, `llm`, `embedder`, `vector_store`

### 8.11 插件配置 (plugins)

- **天气** (`get_weather`)：`api_host`, `api_key`, `default_location`
- **新闻** (`get_news_from_chinanews`/`get_news_from_newsnow`)：RSS/API地址
- **HomeAssistant** (`hass_*`)：`devices`(房间,设备名,entity_id), `base_url`, `api_key`
- **音乐** (`play_music`)：`music_dir`, `music_ext`, `refresh_time`
- **知识库** (`search_from_ragflow`)：`description`, `base_url`, `api_key`, `dataset_ids`

### 8.12 声纹识别配置 (voiceprint)

| 配置项 | 说明 |
|--------|------|
| `url` | 声纹接口地址 |
| `speakers` | 说话人配置，格式：`speaker_id,名称,描述` |
| `similarity_threshold` | 相似度阈值，范围 0.0-1.0，默认 0.4 |

### 8.13 提示词配置

| 配置项 | 说明 |
|--------|------|
| `prompt` | 系统提示词，定义 AI 助手的角色和行为 |
| `prompt_template` | 提示词模板文件名 |
| `system_error_response` | 系统错误时的回复 |
| `end_prompt.enable` | 是否开启结束语 |
| `end_prompt.prompt` | 结束语提示词 |

### 8.14 其他配置

- **MCP接入点** (`mcp_endpoint`)：`ws://mcp接入点ip:端口号/mcp/?token=xxx`
- **上下文源** (`context_providers`)：`url`, `headers`
